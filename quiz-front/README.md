# Quiz Frontend (React + Vite)

## Description

Frontend application for Quiz platform built with React, TypeScript, Vite, and Tailwind CSS.

## Technologies

- **React 18**
- **TypeScript**
- **Vite**
- **Tailwind CSS**

## Installation
```bash
npm install
```

## Configuration

Create a `.env` file in the root directory:
```env
VITE_URL=http://localhost:3000/quizzes
```

## Running the app
```bash
# development server
npm run dev

# build for production
npm run build

# preview production build
npm run preview
```
